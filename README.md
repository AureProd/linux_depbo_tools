# linux_depbo_tools

Juste a git for simple import of depbo-tools to linux

## How to deploy 

```
$ docker build -t depbo:latest .
```

## Utilisation

- For create pbo : ```makepbo -N <LINK_TO_MISSION_DIRECTORY>```
- For extract pbo : ```extractpbo.sh <LINK_TO_MISSION_PBO_FILE>```
