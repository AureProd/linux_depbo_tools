# linux_depbo_tools

Juste a git for simple import of depbo-tools to linux

## How to setup

```bash
./setup.sh
```

## Utilisation

```bash
# For create pbo
./makepbo.sh <LINK_TO_MISSION_DIRECTORY>

# For extract pbo
./extractpbo.sh <LINK_TO_MISSION_PBO_FILE>
```
